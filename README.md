# Student Authentication Android Application

This project is an Android application developed using Android Studio and Gradle that provides a student authentication system along with basic university-related features. The app allows users to register, log in, and access different sections such as a dashboard, profile, timetable, news, and university events.

The application follows a modular structure using activities and fragments, enabling smooth navigation and a clean user interface. It demonstrates core Android development concepts including user authentication, fragment-based navigation, data handling, and UI design.

## Project Overview
- User registration and login functionality
- Dashboard displaying student-related information
- Profile management using fragments
- Timetable feature for managing academic schedules
- News and university events display
- Fragment-based navigation for better UI organization

## Technologies Used
- Java
- Android Studio
- Gradle
- Android SDK
- Fragments & Activities
- XML for UI design
